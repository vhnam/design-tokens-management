# Semantic Tokens

## Recommended grouping for semantic tokens

```plaintext
semantic/
├── surface/          ← background layers
│   ├── bg
│   ├── bg-subtle
│   ├── bg-muted
│   └── bg-inverse
├── content/          ← foreground / text / icon
│   ├── text
│   ├── text-subtle
│   ├── text-disabled
│   └── text-inverse
├── border/           ← strokes / dividers
│   ├── border
│   ├── border-strong
│   └── border-focus
├── interactive/      ← brand / action colors
│   ├── brand
│   ├── brand-hover
│   └── brand-active
└── feedback/         ← status communication
    ├── success
    ├── warning
    ├── error
    └── info
```
